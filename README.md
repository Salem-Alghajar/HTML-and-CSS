# HTML-and-CSS

## 1. Bootstrap Cinema Project
A responsive movie showtimes website utilizing **Bootstrap 5.3**. 
- Features custom card decks for movies like *Kong: Skull Island* and *Ghost in the Shell*.
- Utilizes Bootstrap info-colored pill badges for screening times.
- Includes a smooth-scroll navigation bar linking to showtimes and deals.

## 2. Colorful Portfolio Website
A unique, high-contrast portfolio built from scratch using **pure HTML and CSS**.
- **Visuals:** Features image filters, rounded corners, and center-aligned profile images.
- **Interactivity:** Includes image overlay fades, video backgrounds with text overlays, and hover effects on paragraphs and navbars.
- **Layout:** Demonstrates two-column layouts and custom-styled data tables with fixed navigation.

### Technical Skills Demonstrated:
- HTML5 Video (Autoplay/Mute/Loop)
- CSS Flexbox Layouts
- Fixed Position Navigation
- CSS Image Filters & Overlays
- Table & Form Styling
